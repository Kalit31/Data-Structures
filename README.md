# Data-Structures

This repository holds the implementation of various data structures in **Java**.
* Arrays
* LinkedLists
* Stacks and Queues
* Binary Trees and BST
* Binary Heap
* Hashing
* Sorting algos
* Trees and Graphs

Also it contatins code implementation of programming approaches namely-
* Recursion
* Divide and Conquer
* Greedy Algorithm
* Dynamic Programming
